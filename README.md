# CS5100-UNO-MCTS
Final project for CS5100. Examining 3 different agent's performance against each other in the card game UNO

This project was created using Python - if there are any missing libraries, please install them via a terminal or command line. To run the appropriate files, please follow the instructions below:

1). Download all of the files within this REPO. Ensure the file structure remains consistent with agents and the subfiles stored within that folder.

2). To run the evaulation done on the Monte Carlo Tree Search (MCTS) agent to examine potential hyperparameter tuning values, run the mcts_evaluation.py file. Results for this process are automatically saved in a csv file called 'mcts_evaluation_results.csv' and plotted immediately.

3). To run the evaluation done on the three agents developed for the matchup winrates, run the evaluation.py file. Results are saved in a csv file called 'evaluation_results.csv'.

3.5) To plot the results obtained from Step 3, run the plot_results.py file.
